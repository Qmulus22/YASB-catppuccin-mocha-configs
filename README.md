# YASB Catppuccin Mocha Liquid Glass

A custom **YASB (Yet Another Status Bar)** configuration inspired by the **Catppuccin Mocha** color palette and a modern **Liquid Glass** aesthetic.

## Features

### 🎨 Catppuccin Mocha Theme

- Mauve accents
- Lavender highlights
- Soft blue workspace indicators
- Consistent dark backgrounds
- High readability text colors

### 💎 Liquid Glass Design

- Transparent backgrounds
- Soft borders
- Rounded corners
- Layered shadows
- Frosted-glass inspired appearance

### ⚡ Lightweight

Designed to remain visually appealing while keeping resource usage minimal.

## Requirements

- Windows 10 / Windows 11
- YASB
- JetBrainsMono Nerd Font

## Installation

1. Install YASB.
2. Install **JetBrainsMono Nerd Font**.
3. Copy the files into your YASB configuration directory.

```text
config.yaml
styles.css
```

4. Restart YASB or reload the configuration.

## Customization

Modify the Catppuccin colors in `styles.css`.

```css
--mauve: #cba6f7;
--lavender: #b4befe;
--blue: #448fff;
--red: #f38ba8;
--yellow: #ffd16d;
```

Adjust transparency:

```css
--bg-color1: rgba(39, 38, 39, 0.98);
--bg-color2: rgba(43, 43, 43, 0.98);
```
